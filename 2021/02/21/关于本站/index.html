关于本站

关于我

我的名字和域名一样，但是请叫我曜，现阶段是一名非计算机专业在读大学生，喜欢折腾各种新事物，一心只想好好学习。本人目前对于 C、C++、Java、Python、PHP、node.js、JavaScript、HTML/CSS、MATLAB、Visual Basic、PL/SQL 等编程语言全部精通全不精通, 并且已经完全掌握各国语言 Chinese、English、German、French、Russian、Japanese、Spanish、Italian、Greek、Latin、Polish、Dutch、Arabic、Hindi、Korean、Malay 的单词拼写、中文释义及正确发音。

本站概况

本站基于 Hexo 强力驱动，并搭建其框架,曾经部署于国外 GitHub 和 Coding，由于搭建的网站处于国外，并没有在国内备案，网站处于半墙状态。本打算直接用国外服务器，以免在国内备案，但每年60多刀约400元左右的必要消费起来还是有些肉痛，而且服务器在国外就意味着必须得在国内要做一些优化。但是现在国外舆论攻势等因素，中国互联网处于戒严状态，防火长城墙高不断增加，算下来付出的精力和成本并不划算。

现已暂时部署至阿里云，（通过学生认证利用阿里云高校计划领了两个月的服务器）默默发出白嫖的声音：我，学生，免费，懂?

ps:花钱是不可能花钱的，这辈子都不可能花钱的，白嫖真的很开心，超喜欢白嫖的。

建站缘由

千删鸟飞绝，万禁人踪灭

只想搭建一个属于我自己的世界，可以存放文字和回忆且不受任何影响的地方。

大脑容量有限，有输入才会有输出，用博客记录无尽的知识。

用输出倒逼自己输入，进行深度学习总结和归纳。

–Begin With The End In Mind

记录生活，记录一切，终身学习。

人，应该有所始且有所终，

以终为始，寻找我这一生的目的；

当我意识到的时候，却已经错过了开始；

开始我已记不清了，但终点我还未曾到达；

在这一秒已然此刻不经意地流逝，下一秒却也早已成了此时；

有人喜欢记忆的随风而逝，而我却试图记录下脑海中的全部；

过去成就现在，现在定义将来，或许我这一生将会是一本书；

记录的情节老套，无非是生老病死；

但珍贵的，就是那易逝的记忆。

框架已经搭建完成，后续框架功能会不断慢慢完善。

2021-2-13 材料被打回，申请不予通过

工信部2020年3月1日开始执行了新的《网络信息内容生态治理规定》其中有一条对个人备案网站的规定，就是个人备案的网站不能做博客了。

2021-2-14 重新提交申请，自己整理了一下相关规定 ，在灰色的边缘疯狂试探。

个人备案不能做博客，不能出现“博客”字眼，不能有”评论“、“登录”等功能。

个人备案网站名必须包含备案时填写的网站名，必须完整，建议后跟网站备案名。

主体下所有备案域名必须解析到主体所在的服务商的服务器IP。

控制页面内容，不要出现违法关键词。

域名持有人与备案主体人信息一致。

2021-2-20 备案审核通过

等有钱还是入香港服务器，太特么折腾了，辛酸史就不细说了。
但还是成功拥有属于自己的个人网站，并且也在慢慢美化。
